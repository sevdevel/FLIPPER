# FLIPPER
R-files associated with the 'FLexible Interpretation of Porewater Profiles and Estimation of Rates'

# Required packages
FLIPPER requires the following packages to be installed
- marelac
- shape
- signal
- fractaldim
- abind
- tcltk
- ReacTran
- rootSolve
- deSolve
- FME
- coda
- wavelets
